# MARKOV.OS // TASK-03

### **Description**
Statistical engine utilizing Markov Chains to predict word-to-word transition probabilities based on user-provided corpus. Features a reactive Neo-Brutalist frontend with zero-latency local processing for real-time text synthesis. Secure, efficient, and fully compliant with Prodigy Task-03 requirements.

## **🚀 Key Features**
* **Local Processing:** No external API dependencies; logic runs entirely on the Flask server.
* **N-Gram Probability:** Uses a statistical model to predict the next token based on current state frequency.
* **Gen-Z Aesthetics:** High-end glassmorphism UI with reactive loading states and mesh gradients.
* **Variable Depth:** Supports short, medium, and long generation via iterative sentence stitching.
